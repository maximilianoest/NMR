This script calculates the dependency of the R1 rate with respect to the fibre orientation.
The angles phi and theta in the paper are called position aorund the fibre and fibre orientation, respectively.

Configuration:
(in thist section the not directly understandable configuration parameters are explained)
- fileName: the name of the file that can be found in the data folder. No .mat extension needed!
- dataFieldName: how the data of interest in the file behind fileName is saved.

- there are paths given for local machine and server for easier changes from local machin to server
